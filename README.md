# GolangAsciiArt
## Usage

<pre>Usage of ./asciiImage:
  -contrast int
    	Contrast value to apply to image
  -file string
    	Name of file to turn into ascii art
  -html string
    	print to html file
  -out string
    	name for output to be written
  -print
    	print art to stdout
  -scale float
    	Scaling factor to resize the image (default 1)
</pre>
